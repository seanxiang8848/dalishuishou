## 1️⃣ 抖音来客开放平台 ⭐ 最高优先级

### 申请地址
https://life.douyin.com/openapi

### 前置条件
**必须先完成**：
1. 商家入驻抖音来客（https://life.douyin.com/）
2. 完成抖音来客企业认证
3. 店铺已上线并有正常经营的团购商品

### 需要准备的资料
- [ ] 抖音来客商家账号（已入驻）
- [ ] 营业执照（与来客认证一致）
- [ ] 法人身份证
- [ ] 应用名称：大力水手配送
- [ ] 应用用途说明：桶装水配送管理系统，用于同步抖音来客订单、分配配送员、完成订单核销

### 申请步骤
1. 登录抖音来客商家后台
2. 进入「应用中心」→「应用管理」
3. 点击「创建应用」
4. 填写应用信息：
   - 应用名称：大力水手配送
   - 应用类型：配送管理工具
   - 回调地址：`http://你的服务器/api/douyin/callback`
5. 申请接口权限：
   - ✅ 订单列表查询
   - ✅ 订单详情查询
   - ✅ 核销码核销
   - ✅ 订单状态同步
6. 提交审核，等待3-7工作日

### 申请成功后获得
- **App ID**: 例如 `cli_xxxxxxxxxxxxxxxx`
- **App Secret**: 例如 `xxxxxxxxxxxxxxxx`
- **Access Token**: 用于调用API
- **权限**: 订单读取、核销码核销

### 配置到项目
```
文件: backend/.env
DOUYIN_APP_ID=你的AppID
DOUYIN_APP_SECRET=你的AppSecret
```

### 费用说明
- **API调用**：免费
- **QPS限制**：默认100次/秒（足够使用）

### 注意事项
⚠️ **必须先入驻抖音来客**才能申请开放平台权限
⚠️ 一个抖音来客账号可创建多个应用
⚠️ 审核期间可先用「手动录入订单」功能

## 2️⃣ 高德地图开放平台 ⭐ 最高优先级

### 申请地址
https://console.amap.com/

### 需要准备的资料
- [ ] 个人/企业实名认证
- [ ] 应用名称：大力水手配送
- [ ] 应用类型：出行/物流
- [ ] 包名：com.seanxiangchao.orders

### 申请步骤
1. 注册高德开发者账号
2. 完成实名认证（个人或企业）
3. 创建新应用
4. 添加 Key（选择"Android平台"）
5. 填写包名和 SHA1 证书指纹

### 申请成功后获得
- **Key**: 例如 `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### 配置到项目
```
文件: backend/.env
GAODE_KEY=你的高德Key

文件: android/app/src/main/AndroidManifest.xml
<meta-data
    android:name="com.amap.api.v2.apikey"
    android:value="你的高德Key" />
```

### 费用说明
- **免费额度**：日1万次定位 + 日1万次路径规划
- **付费版**：超出后约 2-5元/千次
- **当前业务量**：免费版完全够用
