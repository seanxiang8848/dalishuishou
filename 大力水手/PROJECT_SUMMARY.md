## 📱 技术架构

- **前端**: Kotlin + Jetpack Compose
- **后端**: Node.js + Express + MySQL
- **网络**: Retrofit + OkHttp
- **图片**: Coil
- **扫码**: ZXing
- **推送**: 极光推送
- **地图**: 高德地图（等待API）
